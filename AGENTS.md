# AGENTS.md

## Goal
Build a simple elegant restaurant website using Astro + Tailwind.

## Pages
- Home
- Menu
- Contact
- Reservations
- Catering

## Requirements
- Mobile-first
- Warm Mediterranean look
- Reusable layout and components
- Netlify Forms for Contact and Catering
- OpenTable embed area on Reservations page
- Clean, simple, production-ready code
- No unnecessary libraries

## Rules
- Plan before coding large changes
- Reuse components
- Keep styling consistent
- Run build checks after major changes
